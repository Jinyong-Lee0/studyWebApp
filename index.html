HTML Tags 속성 설명
<!-- --> annotation 주석을 정의합니다
<h1> - <h6> text semantic HTML 제목을 정의합니다.
<br> text 단일 줄바꿈을 정의합니다.
<p> text 단락을 정의합니다
<em> text 강조된 텍스트를 정의합니다.
<span> text 문서의 섹션을 정의합니다.
<strong> text 중요한 텍스트 정의
<a> text link inline 하이퍼링크를 정의합니다
<abbr> text semantic 약어 또는 두문자어를 정의합니다.
<address> text semantic 문서 작성자/소유자의 연락처 정보를 정의합니다.
<div> block 문서의 섹션을 정의합니다.
<area> block 이미지 맵 내부의 영역을 정의합니다.
<hr> block 콘텐츠의 주제 변경을 정의합니다.
<article> block semantic 기사를 정의합니다
<aside> block semantic 페이지 콘텐츠 이외의 콘텐츠를 정의합니다.
<footer> block semantic 문서 또는 섹션의 바닥글을 정의합니다.
<header> block semantic 문서 또는 섹션의 헤더를 정의합니다.
<main> block semantic 문서의 주요 내용을 지정합니다.
<nav> block semantic 탐색 링크를 정의합니다.
<section> block semantic 문서의 섹션을 정의합니다.
<ol> block list 정렬된 목록을 정의합니다.
<ul> block list 정렬되지 않은 목록을 정의합니다.
<li> block list 목록 항목을 정의합니다.
<audio> block media 포함된 사운드 콘텐츠를 정의합니다.
<form> form 사용자 입력을 위한 HTML 양식을 정의합니다.
<label> form <input> 요소에 대한 레이블을 정의합니다.
<optgroup> form 드롭다운 목록에서 관련 옵션 그룹을 정의합니다.
<option> form 드롭다운 목록에서 옵션을 정의합니다.
<select> form 드롭다운 목록을 정의합니다.
<textarea> form block 여러 줄 입력 컨트롤(텍스트 영역)을 정의합니다.
<input> form inline-block 입력 컨트롤을 정의합니다.
<button> form inline-block 클릭 가능한 버튼 정의
<img> media 이미지를 정의합니다
<video> media 포함된 비디오 콘텐츠를 정의합니다.
<table> table 테이블을 정의합니다
<tbody> table 본문 콘텐츠를 테이블로 그룹화합니다.
<td> table 테이블의 셀을 정의합니다.
<tfoot> table 바닥글 콘텐츠를 표로 그룹화합니다.
<th> table 테이블의 헤더 셀을 정의합니다.
<thead> table 헤더 내용을 테이블로 그룹화합니다.
<tr> table 테이블의 행을 정의합니다.
<blockquote> text block 다른 소스에서 인용된 섹션을 정의합니다.
<iframe> 외부 페이지를 가져옵니다.
