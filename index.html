Javascript 개념을 잡아보자
  1. 변(상)수 설정

    자바스크립트에서 변수는 let, 상수는 const를 사용하여 선언한다.
    let 변수명 = 값; 형태를 사용한다.
    let을 사용한 변수는 값을 바꿀 수 있다. 변수명 = 값 형태를 사용해서.
    그러나 const를 이용해서 값을 지정한 경우, 향후 변경이 불가능하다. 보통은 ID와 같은 변경해서는 안 되는 것들에 넣는다.

  2. 조건문

    자바스크릡트에서 조건문은 if(조건문) 형태를 사용한다.
    예시 : if(i === 0) { console.log("참") }
    i가 0일 때, console에 "참"이라는 값을 띄워준다.

  3. 함수

    자바스크립트에서 함수는 function 함수명(매개변수) { }과 같이 작성한다.
    함수는 수학에서 나오는 함수와 같은 개념으로, 특정 값을 전달받으면 함수 안에 있는 이벤트를 발생시킨다.
    위 함수는 a와 b를 더한 값을 반환해준다.

  4. 반복문

    자바스크립트에서 반복문은 for(let i = 0; i < 5; i++) { console.log(i) }와 같이 작성한다
    일반적으로 반복해야 할 값이 많이 있는 경우에 사용한다.
